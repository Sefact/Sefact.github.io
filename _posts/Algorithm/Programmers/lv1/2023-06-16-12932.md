### 문제 설명

<p>자연수 n을 뒤집어 각 자리 숫자를 원소로 가지는 배열 형태로 리턴해주세요. 예를들어 n이 12345이면 [5,4,3,2,1]을 리턴합니다.</p>

<h5>제한 조건</h5>

<ul>
<li>n은 10,000,000,000이하인 자연수입니다.</li>
</ul>

<h5>입출력 예</h5>

| n     | return      |
|-------|-------------|
| 12345 | [5,4,3,2,1] |

### 정답

```python
def solution(n):
    return [int(i) for i in str(n)[::-1]]
```
