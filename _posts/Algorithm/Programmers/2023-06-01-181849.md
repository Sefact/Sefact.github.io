### 문제 설명

<p>한 자리 정수로 이루어진 문자열 <code>num_str</code>이 주어질 때, 각 자리수의 합을 return하도록 solution 함수를 완성해주세요.</p>

<hr>

<h5>제한사항</h5>

<ul>
<li>3 ≤ <code>num_str</code> ≤ 100</li>
</ul>

<hr>

<h5>입출력 예</h5>

| num_str     | result |
|-------------|--------|
| "123456789" | 45     |
| "1000000"   | 1      |

<hr>

<h5>입출력 예 설명</h5>

<p>입출력 예 #1</p>

<ul>
<li>문자열 안의 모든 숫자를 더하면 45가 됩니다.</li>
</ul>

<p>입출력 예 #2</p>

<ul>
<li>문자열 안의 모든 숫자를 더하면 1이 됩니다.</li>
</ul>

### 정답

```python
def solution(num_str):
    answer = 0
    
    for num in num_str:
        answer += int(num)
    
    return answer
```
